YouTube Video Downloader
A simple GUI-based application to download YouTube videos. This application is built using Python and the following libraries:

customtkinter for the GUI
pytube for downloading YouTube videos
PIL for image processing
pyperclip for clipboard operations

Project Structure
main.py: The main script containing the application code.

Features
Paste YouTube URL directly from the clipboard.
Clear the URL entry field.
Select video resolution (1080p, 720p, 360p, 240p, 144p).
Display download progress.
Save videos to the user's Downloads directory.
